# TK8Info

A small Android reference app I built for myself — a quick way to glance at each Tekken 8 character's sidestep direction, throw-break category, and the stances/installs/heat states they have access to.

It's a personal notes app. I'm not planning to put it on the Play Store, and I don't intend to maintain it for anyone but me. The data in here reflects what I happened to care about remembering; some entries are filled in and many are still placeholders.

If any of this is useful to you, feel free to fork it and extend it however you like — add the characters and states you care about, swap the data model, redesign the UI, ship your own version. There's no expectation that contributions flow back upstream. Pull requests aren't being actively reviewed.

The character portrait sprite sheet comes from [wavu.wiki](https://wavu.wiki/). If you publish a fork, make sure your use of that asset is compatible with their license.

For build instructions, dependency versions, and architectural notes (Nav3 + Material3 adaptive list-detail, sprite-sheet slicing, the per-character data layout), see [CLAUDE.md](CLAUDE.md).
